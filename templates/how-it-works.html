How Does the Application Work?

If you have multiple .mp3 files and want to merge them into groups (e.g., in pairs of 2):

Follow these steps:

  1. Upload the .mp3 files you want to merge.
  2. Select the number of files per group (default: all files will be merged into one).
  3. Click the merge button and wait for the process to complete.
  4. Once done, you will receive a .zip archive containing your merged files.
